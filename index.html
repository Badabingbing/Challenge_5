AUDIOTHEME FRAMEWORK DEMO

The Audio Theme Framework plugin is built to work with any WordPress theme, providing the features need for a music-oriented website and allowing you to take your content with you anywhere you go -even when switching themes.

In this demo, we're showcased how the Audio Theme Framework will look with the Twenty Fourteen theme and a little help with this child theme we've created. Take a look around at the functionality for displaying gigs ans venues, discography (liner notes for the web), and videos.
